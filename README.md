## 주요 기능 

- 로그인 / 회원가입
- 게시글 작성 및 조회
- 채팅 및 사진 전송
- 카테고리 별로 조회 
- 마이페이지


<br>
<br>

<h2>Android Tech Stack</h2>

- Minumum SDK 23
- Firebase RealtimeDB
- Firebase Auth
- Kotlin
    - Coroutines
- Android Jetpack
    - Navigation Component
    - ViewBinding
- Material Components
